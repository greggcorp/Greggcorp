This website runs on [Jekyll](https://jekyllrb.com/), using a very Frankensteined version of the [Joseph](https://jekyllthemes.io/theme/joseph-blog-jekyll-theme) theme by Artem Sheludko. The site is built and deployed by [Netlify](https://www.netlify.com/).

The type is set in [Inter](https://fonts.google.com/specimen/Inter?query=inter) by Rasmus Andersson and is served by [Google Fonts](https://fonts.google.com/).

Icons throughout the site are served by [Font Awesome](https://fontawesome.com/).

This site uses RSS to share new posts to LinkedIn and Twitter via [Zapier](https://zapier.com/), and to Mastodon via [IFTTT](https://ifttt.com/). New content is also shared via RSS to [Buttondown](https://buttondown.email/) to create newsletter updates.
